3. 핵심 기능
이 서비스의 핵심 기능은 관객의 표정을 분석하여 점수화하는 것입니다.
사용자는 관객의 표정을 업로드하고 예상하는 감정을 입력하면 실제 감정과 차이를 점수화합니다.  

4. 핵심 트러블 슈팅
      
   4.1. 표정 분류의 정확성
  주어진 데이터셋은 분노, 행복, 슬픔, 공포 4가지인데 영화를 보는 관객의 감정은 그 이상이고 데이터셋의 라벨링에도 문제가 있어 무감정을 추가하여 재라벨링하는 과정에서 정확성이 매우 떨어졌습니다.
  앙상블을 통하여 정확도를 높였는데 데이터의 부족으로 주어진 데이터론 88%, 필드데이터론 80%의 정확도를 보여 정확성을 추가로 높일 필요가 있습니다.  

   4.2 다양한 표정 분석의 필요성
  영화를 보며 느낄 만한 감정은 다양하고 제작자가 원하는 감정도 다양할텐데 기존 감정을 분석하는 tool과 주어진 데이터셋의 표현 감정은 매우 제한적이었습니다. 
  그래서 기존 감정에서 분석한 데이터를 조합하여 영화를 보며 느낄 만한 감정을 추출했습니다.
  이 과정에서 여러 논문과 심리학 이론을 기반으로 각 감정을 벡터화하였습니다.
